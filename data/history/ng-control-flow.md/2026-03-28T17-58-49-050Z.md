# Angular Control Flow

---

#### `@if`
```html
<div class='result container'>
  @if( win() ) {
    <p>You win</p>
  }
  @else if( lose() ) {
    <p>You lose</p>
  }
  @else {
    <p>Draw</p>
  }
</div>
```

---

#### `@for`
```typescript
items = signal([
  {title: 'one', id: 1}, {title: 'two', id: 2}, 
  {title: 'three', id: 3}, {title: 'four', id: 4}
])
```

```angular181html
@for(item of items(); track item.id) {
  <p>{{ item.title }}</p>
}
```

<ul>
    <li class="fragment"><code>track</code> muss ein Ausdruck folgen, welcher für jedes item unique ist</li>
    <li class="fragment"><code>track $index</code> einfach solang keine Neuordnung</li>
</ul>

---

```angular181html
@for(item of items; track $index) {
    {{ $index }}: {{ item }}
}
@empty {
    <p>Please add some items</p>
}
```
<!-- .element: class="fragment" -->

```text
$count => items.length
```
<!-- .element: class="fragment fade-in-then-semi-out" -->

```text
$first => $index === 0
```
<!-- .element: class="fragment fade-in-then-semi-out" -->

```text
$last  => $index === $count - 1
```
<!-- .element: class="fragment fade-in-then-semi-out" -->

```text
$even  => $index % 2 === 0
```
<!-- .element: class="fragment fade-in-then-semi-out" -->

```text
$odd   => !$even
```
<!-- .element: class="fragment fade-in-then-semi-out" -->

#### `@switch`
```angular181html
@switch (state()) {
  @case ('login') {
    <app-login></app-login>
  }
  @case ('shop') {
    <app-shop></app-shop>
  }
  @case ('admin') {
    @if (isAdmin(user)) { 
      <app-admin></app-admin> 
    }
  }
  @default {
    <app-not-found></app-not-found>
  }
}
```
